POST
/api/v1/reports/{company_id}/{report_id}/documents
Add Report Documents


Attach more documents to an existing report.

Scope (year, sector, frameworks, regulators) is read from the report's stored generation_config — the caller never re-enters params. Files go through the same worker pipeline as /generate, with report_id threaded so every downstream row (documents, digital_twin_states, esg_evidence) is stamped with this report.
